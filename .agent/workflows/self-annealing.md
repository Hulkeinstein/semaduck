---
description: How to self-correct system errors and refine directives (Self-Annealing Procedure)
---

1. Analyze: Check `main.py` logs or execution error messages to identify the Root Cause (RC).
2. Fix: Modify Python scripts in `execution/` or adjust environment variables.
3. Verify: Execute the modified tool independently to confirm normal operation.
4. Update: Update the corresponding directive in `directives/` with discovered constraints (API limits, edge cases, etc.) to enhance intelligence.
5. Report: Report modifications and improvements to the user in Korean.
