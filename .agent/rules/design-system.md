# Rule: Design System

## 1. Color Palette
- **Cream**: `#FAF9F6` (Primary Background)
- **Brown**: `#2D2926` (Primary Text/Accents)
- **Gold**: `#C5A039` (Highlights)

## 2. Typography
- **Heading**: `Playfair Display` (Serif, Elegant)
- **Body**: `Manrope` or `Noto Sans KR` (Sans-serif, Clean)

## 3. Visual Stylings
- **Rounding**: Max `2px` for a sharp, premium feel.
- **Animations**: Use `framer-motion` for subtle, slow entries.
- **Aesthetics**: Glassmorphism, premium gradients, and high-quality imagery.

## 4. Constraints
- No default browser colors.
- All interactive elements must have hover states.
